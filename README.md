# Rambam Learning Engine v2

A learning system for mastering the structure of the Mishneh Torah.

## Learning hierarchy

1. 14 Sefarim
2. 83 Halachic Sections
3. 613 Mitzvot
4. Chumash Sources

## Rambam's Phone Number

5, 6, 10, 5, 3, 4, 7, 9, 5, 8, 6, 5, 5, 5

Total: 83 sections

## Current project convention

This project follows the 83-section learning structure used in this learning system.

In cases where later printed editions or secondary sources count material differently (for example, Seder Tefilot at the end of Ahavah), the structure used here follows the framework adopted for this project.
